JS, SCSS working set using Esbuild for my personal usage


## Usage

* Build
```powershell
$ make          # or mingw32-make.exe
```

* Print help
```powershell
$ ./builder -h
```

* Generate asset files
```powershell
$ ./builder -t
$ yarn          # or npm install
```

* Watch
```powershell
$ ./builder -w
```

* Build
```powershell
$ ./builder
```
